fetch() - idhu oru main ana method , idha use pani dhan backend data va frontend la show panuvanga 

fetch() kulla pora munna namaku 3 visiyam theriyanum 

1. API (Application Programming Interface) 
    1. API na oru Webservice 
    2. API vachu namma 2 websites ah intract panikalam , Data va share pandratha vachu communicate pannikum 
    For Eg : Google Weather API  
    Sample API : 

    There are many type : 

    RESTAPI - Representation State Transfer
    SOAP API - Simple Object Access Protocal 

2. HTTPS Request

    1. API la ulla data va edukurathukum update pandrathukum nama Andha API Web Service ku HTTPS Request anupanum 
    2. Andha Request correct ah irundha data JSON Format la varum ilana browser oru error ah throw pannum 

3. JSON ()
    JSON - Javascript Object Notation 
    1. Idhula data ellam JSON format la varum, adha namma client side use pana indha method use agum 
    2. Return agurah data JSON ah irukalam Or else Xml ah irukala
